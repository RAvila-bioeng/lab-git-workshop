# Lab Git Workshop

This is a simple practice repository so my team can learn the basics of Git and GitHub.

## Goal

The goal of this workshop is to help everyone practice a basic collaborative workflow:

1. Clone the repository
2. Open it in Visual Studio Code
3. Edit a file
4. Stage the changes
5. Commit the changes
6. Push them to GitHub

## Files in this repo

- `README.md` → instructions for the workshop
- `books.csv` → a simple dataset where everyone can add one book

## Exercise

Each person should:

1. Open `books.csv`
2. Add **one new row** with a book they like
3. Save the file
4. Stage the change
5. Commit with a message like:

```bash
git commit -m "Add my book"
